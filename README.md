# SI-Rawat-Jalan
sistem informasi rawat jalan adalah sebuah sistem informasi yang dibangun dengan menggunakan vb.net 2010 dan dengan database ms acces. 
fitur-fitur yang ada adalah antara lain: 
  + Data Poli
  + Data Dokter
  + Data Pasien
  + Data Pemakai / User
  + Data Obat
  + Data Pendaftaran
  + Data Resep
